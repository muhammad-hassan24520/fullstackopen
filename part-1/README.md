# Part 1
## Part 1 Summary

During Part 1 of Full Stack Open, I learned the core concepts of React and modern JavaScript by building interactive web applications.

* **React Components** – Created reusable UI components to build modular applications.
* **JSX** – Used HTML-like syntax to describe React user interfaces.
* **Props** – Passed data between components efficiently.
* **Component Structure** – Organized applications into small, maintainable components.
* **JavaScript ES6+** – Practiced variables, arrays, objects, functions, destructuring, and the spread operator.
* **State Management** – Managed dynamic data using the `useState` hook.
* **Event Handling** – Responded to user interactions through event handlers.
* **Conditional Rendering** – Displayed content based on application state.
* **Arrays & Objects in State** – Updated complex state immutably following React best practices.
* **Component Re-rendering** – Understood how state changes update the user interface.
* **Application Development** – Built the Course Information, Unicafe, and Anecdotes applications to apply React fundamentals in real-world scenarios.

Part 1 established a strong foundation in React development and prepared me for more advanced concepts in the upcoming sections of the Full Stack Open course.
